Detailed Description

Abstract base class for all Hue sensors.

Sensors are ObservableObjects. Subscribing to OnStateChanged gives real-time property-change notifications whenever the bridge reports a new reading via SSE.

Definition sensor_base.h:136

Get or set the sensor's display name.

Assigning a new value PUTs {"metadata":{"name":"\u2026"}} to the bridge immediately. The bridge confirms the rename via an SSE event which updates the backing field and fires PropertyChanged a second time.

Exceptions
InvalidParameterExceptionif the name is empty
BridgeNotReachableExceptionif the bridge is not available
AuthenticationExceptionif not authenticated
