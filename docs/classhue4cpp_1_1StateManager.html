Detailed Description

Manages state synchronization via Server-Sent Events.

The StateManager maintains an up-to-date view of the bridge state by listening to SSE events from the bridge.

◆ OnResourceEvent

ReactiveLitepp::Event<const ResourceEventArgs&> hue4cpp::StateManager::OnResourceEvent

Single unified event fired for every resource change.

Covers lights (added / changed / removed), sensors (added / changed / removed) and bridge connectivity changes. Use ResourceEventArgs::type or the helper predicates (isLightEvent, isSensorEvent, isBridgeEvent) to filter the events you care about.

auto sub = state_manager.OnResourceEvent.SubscribeScoped(
[](const ResourceEventArgs& e) {
if (e.isLightEvent()) { ... }
if (e.isSensorEvent()) { ... }
if (e.isBridgeEvent()) { ... }
});
